La aplicación desarrollada para la empresa **Dogix** tiene como objetivo gestionar información relacionada con perros y los servicios que ofrece la empresa. Dogix es una empresa dedicada exclusivamente al cuidado, atención y gestión de servicios para perros.

El sistema funciona mediante una **API**, la cual permite la comunicación entre la aplicación y la base de datos. Para el almacenamiento de la información se utiliza **MySQL**, permitiendo guardar de forma organizada los datos de los perros, clientes y servicios.

La conexión entre la API y la base de datos se realiza a través de una infraestructura desplegada en **Render**, lo que facilita el acceso remoto, la estabilidad del servicio y la administración del sistema en línea.

Esta aplicación busca optimizar la gestión de la información dentro de Dogix, permitiendo un control eficiente de los registros y mejorando la organización de los servicios dedicados al cuidado de los perros.
